HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML attributes are always defined in the beginning element never the closing element
  • Elements that are nested inside other elements are called Child Elements
  • HTML elements work together to build the structure of a web page and provide a framework that makes it easy for a user to navigate the webpage.
  • HyperText Markup Language is one of the three cornerstone languages used in web development.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style a page using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • The most common way to style a webpage is by using an external CSS style sheet.
  • Cascading Style Sheets is a programming language used by developers to define how webpage content is presented to users.
  • CSS defines the colors, aesthetics, visual layout, and organization of a webpage.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git is a version control system that lets you manage and keep track of each version of your project
  • The git command must precede every Git statement
  • git pull origin main is the git pull command to receive a branch's modifications into the local environment
  • git push orirgin is the git command to push the changes we have locally to our remote GitHub branch
